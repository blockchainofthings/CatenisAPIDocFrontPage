The Catenis Enterprise API

The Catenis Enterprise API is the means by which the end user interacts and uses the Catenis Enterprise system.

It makes Catenis Enterprise's functionality available as a Web service, so it can be easily accessed from anywhere an Internet connection is available, and from different programming environments.

API reference documentation

The Catenis Enterprise API reference documentation is where technical people can find detailed information on the available functionality and how to properly use it.

The reference documentation is organized by API versions, being version 0.6 the latest available one.

Change log

The change log section describes the changes made to a new version of the API from the immediately previous version.
