Overview

The Catenis Enterprise is an integration layer to the global bitcoin blockchain. It allows companies to rapidly build blockchain based applications or simply integrate with existing systems.

This integration layer is made available via a RESTful web services API, which is how end users (developers) interact and use the Catenis Enterprise system.

The API exposes all of Catenis Enterprise's powerful functionality so it can be easily accessed from a myriad of programming languages and environments.

API reference documentation

The Catenis Enterprise API reference documentation is where technically savvy people can find detailed information on the available functionality and how to properly use it.

The reference documentation is organized by API versions, with version 0.6 being the latest available.

The change log section describes the updates and modifications made to a new version of the API from its immediate previous version.
